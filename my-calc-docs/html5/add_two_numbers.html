Add two numbers

Get addition result using the CALC app.

This task explains how to add two numbers using the CALC app.

  1. Enter the first number.
  2. Select the plus (+) button.
  3. Enter the second number.
  4. Select the equals (=) button.
The result from the addition is displayed on the screen.

Adding two numbers

To add two numbers, enter 3 + 5. The result is 8.
