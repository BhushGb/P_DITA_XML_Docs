Multiply two numbers

Get multiplication result using the CALC app.

This task explains how to multiply two numbers using the CALC app.

  1. Enter the first number.
  2. Select the multiplication (×) button.
  3. Enter the second number.
  4. Select the equals (=) button.
The result from the multiplication is displayed on the screen.

Multiplying two numbers

To mulitply two numbers, enter 2 × 3. The result is 6.
