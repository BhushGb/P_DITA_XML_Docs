Divide two numbers

Get division result using the CALC app.

This task explains how to divide two numbers using the CALC app.

  1. Enter the first number.
  2. Select the division (÷) button.
  3. Enter the second number.
  4. Select the equals (=) button.

The result from the division is displayed on the screen.

Dividing two numbers

To divide two numbers, enter 6 ÷ 3. The result is 2.

Note:

You cannot divide a number by zero. The CALC app displays an error when the second number is 0.
