Subtract two numbers

Get subtraction result using the CALC app.

This task explains how to subtract one number from another using the CALC app.

  1. Enter the first number.
  2. Select the minus (-) button.
  3. Enter the second number.
  4. Select the equals (=) button.
The result from the subtraction is displayed on the screen.

Subtracting two numbers

To subtract two numbers, enter 7 − 3. The result is 4.

If the first number is smaller than the second number (for example, 3 − 7), the result is a negative number. In this case, −4 is displayed on the screen.
